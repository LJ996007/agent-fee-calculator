# Agent Fee Calculator (Vite + React + TypeScript + Tailwind)

## 快速开始
1. 安装 Node.js (建议 18+)
2. 在项目目录执行：
   ```bash
   npm install
   npm run dev
   ```
3. 浏览器打开提示地址（通常为 http://localhost:5173/ ）

## 生产构建
```bash
npm run build
npm run preview
```

## 说明
- 图标库：`lucide-react`
- 样式：`tailwindcss`
- 你的页面代码在 `src/AgentFeeCalculator.tsx`
